# Retail Analysis

## Problem Statement
The objective of this project is to analyze retail inventory and sales data to identify stock issues, improve inventory efficiency, and support better business decisions.

---

##  Dataset
This **Retail Store Inventory and Sales Dataset** includes:
- Product categories  
- Inventory levels  
- Pricing details  
- Units sold (sales data)

  Dataset Link : https://www.kaggle.com/datasets/sandhyapeesara/retail-store-inventory

---

## Tools & Technologies
- Power BI

---

## Key Analysis
- Product-wise sales performance  
- Inventory level monitoring  
- Stockout identification  
- Revenue contribution by category  

---

## Key Metrics (KPIs)
- Total Units Sold  
- Total Revenue  
- Inventory Level  
- Stockout Count  
- Inventory Turnover
